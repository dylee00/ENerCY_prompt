# data-schema.md

## 문서 목적

이 문서는 Fuels Futures 데이터셋의 컬럼 구조와 데이터 타입을 정의한다.

---

## 전체 컬럼 구조

| 컬럼명 | 데이터 타입 | 설명 |
|--------|------------|------|
| ticker | str | 종목 코드 (예: CL=F) |
| commodity | str | 오일 종류명 (예: Crude Oil) |
| date | str | 거래일 (YYYY-MM-DD 형식) |
| open | float64 | 시가 (달러) |
| high | float64 | 고가 (달러) |
| low | float64 | 저가 (달러) |
| close | float64 | 종가 (달러) |
| volume | int64 | 거래량 |

---

## 날짜 처리 기준

- `date` 컬럼은 원본에서 `str` 타입으로 저장됨
- 전처리 시 `datetime` 타입으로 변환 후 사용
- 시계열 정렬 기준: `date` 컬럼 오름차순
